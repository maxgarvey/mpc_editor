# cmd

Entry-point binaries for the MPC Editor project.

## Programs

| Directory | Binary | Purpose |
|-----------|--------|---------|
| [`mpc_editor/`](mpc_editor/README.md) | `mpc_editor` | Main web server — starts the HTTP server and opens the browser |
| [`genseq/`](genseq/README.md) | `genseq` | Test fixture generator — writes a curated set of `.SEQ` files to `testdata/seq/` for hardware verification |

## Building

```bash
make build          # builds cmd/mpc_editor → ./mpc_editor
go build ./cmd/...  # builds all binaries
```
